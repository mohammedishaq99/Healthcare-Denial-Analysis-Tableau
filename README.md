# Healthcare Claims Denial Analysis

## Project Overview

This project analyzes healthcare insurance claim denials using Tableau. The dashboard helps identify denial trends, financial impact, payer performance, and claim status to support data-driven decision-making.

## Dataset

- Healthcare Claims Dataset
- Cleaned using Excel/Python
- Format: CSV

## Tools Used

- Tableau
- Excel
- Python 

## Dashboard Features

- Total Claims
- Total Denied Claims
- Approval Rate
- Denial Rate
- Denial Reasons
- Claims by Insurance Provider
- Monthly Claim Trends
- Geographic Analysis

## Business Insights

- Identifies providers with highest denial rates
- Tracks claim approval trends
- Highlights common denial reasons
- Supports operational improvements
